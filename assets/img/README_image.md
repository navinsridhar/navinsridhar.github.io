# Put your headshot here

Save a square JPEG named **`prof_pic.jpg`** in this folder (`assets/img/`).
Recommended: 600–1000 px on a side, under 500 KB after compression.

If you don't have a headshot handy, a placeholder image is fine to start with —
the site will still build. Delete this `README_image.md` file once you've
dropped your actual photo in.
