# Research-page figures

The `_pages/research.md` page renders one figure per research thread. Drop a
JPG/PNG with the filenames below to replace the SVG placeholders that ship
with the repo.

| File                            | Section                                                             | Suggested figure                                                                  |
| ------------------------------- | ------------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| `corona-reconnection.svg/.jpg`  | 1 · Black-hole coronae · reconnection · Comptonization              | Plasmoid / current-sheet snapshot from Sridhar+ 2021 MNRAS 507 or 2025 ApJ 979.   |
| `frb-hypernebula.svg/.jpg`      | 2 · Fast Radio Bursts · persistent radio sources · hypernebulae     | Hypernebula geometry / radio SED from Sridhar & Metzger 2022 ApJ 937.             |
| `merger-precursor.svg/.jpg`     | 3 · Multi-messenger precursors of compact-object mergers            | Shock-front / radio-precursor cartoon from Sridhar+ 2021 MNRAS 501.               |
| `em-pair-plasma.svg/.jpg`       | 4 · Strong electromagnetic waves in pair plasmas                    | PIC snapshot of the wave–plasma interaction from Sridhar+ 2026 (arXiv:2604.11698).|

Once you save a JPG/PNG with one of the basenames above (any extension is fine
as long as you update the `path` in `_pages/research.md`), the SVG placeholder
will be replaced automatically on the next build.
